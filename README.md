# select-star-from-work

A personal collection of essential SQL queries, snippets, and scripts gathered from my daily work — built for reuse, not reinvention.

> 🚧 This repository is actively maintained and will continue to grow with more queries and categories over time.

---

## 📁 Structure

### 🛠️ `db-management/`
Scripts for common database administration tasks — covering backup, restore, copy, rename, and connection management on SQL Server / Azure SQL.

---

### 🔍 `metadata/`
Scripts for exploring database structure and statistics — useful for quickly understanding table schemas and size information.

---

## 🔧 Requirements

- Microsoft SQL Server / Azure SQL Database
- Tested with T-SQL (compatible with SQL Server 2016+)

---

## 🗺️ Roadmap

More scripts are on the way. Planned categories include:

- `performance/` — query tuning, index analysis, execution plan helpers
- `security/` — user management, permission auditing
- `etl/` — data loading, transformation patterns
- `monitoring/` — job history, blocking detection, long-running queries

---

## 📌 Notes

- Scripts are written for **SQL Server / Azure SQL** and use **T-SQL** syntax.
- Always review and test scripts in a non-production environment before running them in production.
- Feel free to adapt these to your own environment.
